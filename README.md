### Create Static Sites in Ruby with Rack

##### Folder Structure
```
.
├── Gemfile
├── Gemfile.lock
├── README.md
├── config.ru
└── public
    ├── css
    ├── images
    ├── index.html
    └── js
```

##### Running locally
```
bundle install
rackup
```
Go to your browser and open http://localhost:9292
